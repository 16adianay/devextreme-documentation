##### shortDescription
A Boolean value specifying whether or not the main screen is inactive while the lookup is active.

---
[note]This option makes sense only if the [usePopover](/api-reference/10%20UI%20Widgets/dxLookup/1%20Configuration/usePopover.md '/Documentation/ApiReference/UI_Widgets/dxLookup/Configuration/#usePopover') option is set to **false**. Otherwise, shading is always disabled.
