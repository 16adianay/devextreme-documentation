##### shortDescription
Specifies a transportation mode to be used in the displayed route.

---
When configuring the widget using [ASP.NET MVC Wrappers](/concepts/35%20ASP.NET%20MVC%20Wrappers/20%20Fundamentals '/Documentation/Guide/ASP.NET_MVC_Wrappers/Fundamentals/'), specify this option using the `GeoMapRouteMode` enum. This enum accepts the following values: `Driving` and `Walking`.
