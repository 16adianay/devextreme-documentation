##### shortDescription
Specifies a transportation mode to be used in the displayed route.

---
When using the widget as an [ASP.NET MVC Control](/concepts/35%20ASP.NET%20MVC%20Controls/20%20Fundamentals '/Documentation/Guide/ASP.NET_MVC_Controls/Fundamentals/'), specify this option using the `GeoMapRouteMode` enum. This enum accepts the following values: `Driving` and `Walking`.
