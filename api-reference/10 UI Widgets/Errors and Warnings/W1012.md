---
##### shortDescription
Occurs in the **DataGrid** widget if the key field cannot be found in at least one data object.

---
Check that all data objects have the field specified in the widget's [keyExpr](/api-reference/10%20UI%20Widgets/dxDataGrid/1%20Configuration/keyExpr.md '/Documentation/ApiReference/UI_Widgets/dxDataGrid/Configuration/#keyExpr') option or the **Store**'s **key** option.
