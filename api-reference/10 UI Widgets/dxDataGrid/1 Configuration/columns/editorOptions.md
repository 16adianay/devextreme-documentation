##### shortDescription
Specifies configuration options for the editor widget of the current column.

---
[note]If you specify the **onValueChanged** handler for the editor, the **DataGrid** will not save the value changed after editing the current cell. Instead, implement the [onEditorPreparing](/api-reference/10%20UI%20Widgets/dxDataGrid/1%20Configuration/onEditorPreparing.md '/Documentation/ApiReference/UI_Widgets/dxDataGrid/Configuration/#onEditorPreparing') handler.
