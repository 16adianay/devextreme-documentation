##### shortDescription
Gets the data objects of currently selected grid records.

##### return: array
The data objects of selected records.

---
To get only the keys of selected grid records, call the [getSelectedRowKeys()](/api-reference/10%20UI%20Widgets/dxDataGrid/3%20Methods/getSelectedRowKeys().md '/Documentation/ApiReference/UI_Widgets/dxDataGrid/Methods/#getSelectedRowKeys') method.
