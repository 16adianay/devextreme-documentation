##### shortDescription
Specifies a custom template for menu items.

##### param(itemData): Object
The current menu item's data.

##### param(itemIndex): Number
The current menu item's index.

##### param(itemElement): dxElement
The current menu item's container. It is an [HTML Element](https://developer.mozilla.org/en-US/docs/Web/API/HTMLElement) or a [jQuery Element](https://api.jquery.com/Types/#jQuery) when you use jQuery.

##### return: String|Node|jQuery
A template name or container.
