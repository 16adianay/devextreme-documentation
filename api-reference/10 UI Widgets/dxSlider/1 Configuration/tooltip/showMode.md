##### shortDescription
Specifies whether the widget always shows a tooltip or only when a pointer is over the slider.

---
When configuring the widget using [ASP.NET MVC Wrappers](/concepts/35%20ASP.NET%20MVC%20Wrappers/20%20Fundamentals '/Documentation/Guide/ASP.NET_MVC_Wrappers/Fundamentals/'), specify this option using the `SliderTooltipShowMode` enum. This enum accepts the following values: `OnHover` and `Always`.
