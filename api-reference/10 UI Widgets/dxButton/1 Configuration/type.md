##### shortDescription
Specifies the button type.

---
DevExtreme provides the following button types.

![Button Types](/images/UiWidgets/Button_Types.png) 

When configuring a widget using [ASP.NET MVC Wrappers](/concepts/35%20ASP.NET%20MVC%20Wrappers/20%20Fundamentals '/Documentation/Guide/ASP.NET_MVC_Wrappers/Fundamentals/'), specify this option using the `ButtonType` enum. This enum accepts the following values: `Normal`, `Default`, `Back`, `Danger` and `Success`.
