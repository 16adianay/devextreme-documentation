##### shortDescription
Specifies item selection mode.

---
When using the widget as an [ASP.NET MVC Control](/concepts/35%20ASP.NET%20MVC%20Controls/20%20Fundamentals '/Documentation/Guide/ASP.NET_MVC_Controls/Fundamentals/'), specify this option using the `NavSelectionMode` enum. This enum accepts the following values: `Single` and `Multiple`.
