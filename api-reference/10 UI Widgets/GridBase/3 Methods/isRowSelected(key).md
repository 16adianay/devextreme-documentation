##### shortDescription
Checks whether the row with a specific key is selected. Takes effect only if **selection**.[deferred](/api-reference/10%20UI%20Widgets/dxDataGrid/1%20Configuration/selection/deferred.md '{basewidgetpath}/Configuration/selection/#deferred') is **false**.

##### param(key): any
The key of the row.

##### return: Boolean
**true** if the row is selected; **false** otherwise.
