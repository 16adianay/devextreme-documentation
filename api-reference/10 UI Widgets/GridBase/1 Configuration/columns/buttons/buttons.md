##### shortDescription

Applies only if the column's [type](/api-reference/10%20UI%20Widgets/dxDataGrid/1%20Configuration/columns/type.md '/Documentation/ApiReference/UI_Widgets/dx{WidgetName}/Configuration/columns/#type') is *"buttons"*.
