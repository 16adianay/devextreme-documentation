##### shortDescription
Specifies text for the item that clears the applied filter. Used only when a cell of the filter row contains a select box.

---
In columns that contain Boolean values or use [lookup](/api-reference/10%20UI%20Widgets/GridBase/1%20Configuration/columns/lookup '{basewidgetpath}/Configuration/columns/lookup/'), a user filters data with a select box. To clear the applied filter, the user selects the *"All"* item in this select box. The **showAllText** option specifies text for this item.
