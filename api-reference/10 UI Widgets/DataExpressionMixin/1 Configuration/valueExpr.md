##### shortDescription
Specifies which data field provides unique values to the widget's [value](/api-reference/10%20UI%20Widgets/DataExpressionMixin/1%20Configuration/value.md '{basewidgetpath}/Configuration/#value'). When the data field is not set, the **value** is the entire data object.

---
Alternatively, you can specify the [key](/api-reference/30%20Data%20Layer/Store/1%20Configuration/key.md '/Documentation/ApiReference/Data_Layer/CustomStore/Configuration/#key') in the store.
