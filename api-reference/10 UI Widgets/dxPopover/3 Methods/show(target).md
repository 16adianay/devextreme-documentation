---
##### shortDescription
Shows the widget for a target element.

##### param(target): String|Node|jQuery
The target element.

##### return: Promise<void>
A Promise that is resolved if the widget is displayed and rejected otherwise. It is a [native Promise](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Promise) or a [jQuery.Promise](https://api.jquery.com/Types/#Promise) when you use jQuery.
