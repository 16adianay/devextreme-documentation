##### shortDescription
Allows an end-user to change filtering options.

---
With this option enabled, an end-user can filter pivot grid data using the filter icons in the [field chooser](/concepts/10%20UI%20Widgets/71%20Pivot%20Grid/010%20Visual%20Elements/10%20Field%20Chooser/01%20Field%20Chooser.md '/Documentation/Guide/UI_Widgets/Pivot_Grid/Visual_Elements/#Field_Chooser') or on the [field panel](/concepts/10%20UI%20Widgets/71%20Pivot%20Grid/010%20Visual%20Elements/15%20Field%20Panel.md '/Documentation/Guide/UI_Widgets/Pivot_Grid/Visual_Elements/#Field_Panel').
