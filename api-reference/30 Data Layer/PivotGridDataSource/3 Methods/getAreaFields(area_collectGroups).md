---
##### shortDescription
Gets all fields within a specified area.

##### param(area): String
[Area](/api-reference/30%20Data%20Layer/PivotGridDataSource/1%20Configuration/fields/area.md '/Documentation/ApiReference/Data_Layer/PivotGridDataSource/Configuration/fields/#area') type.

##### param(collectGroups): Boolean
Specifies whether to display groups as separate items.

##### return: Array
An array of [fields](/api-reference/30%20Data%20Layer/PivotGridDataSource/1%20Configuration/fields '/Documentation/ApiReference/Data_Layer/PivotGridDataSource/Configuration/fields/').

---
If the **collectGroups** parameter is set to **true**, this method will return groups as separate items without going into their structure.
