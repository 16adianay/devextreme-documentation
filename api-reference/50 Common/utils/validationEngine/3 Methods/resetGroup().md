---
##### shortDescription
Resets the values and validation result of the editors that belong to the default validation group.

---
Use this method to reset the validation result for the validation group that combines the editors created by the [jQuery Approach](/concepts/10%20UI%20Widgets/0%20Basics/10%20Widget%20Basics%20-%20jQuery '/Documentation/Guide/UI_Widgets/Basics/Widget_Basics_-_jQuery/') and for which the **validationGroup** option is not specified. This method calls the [reset](/api-reference/10%20UI%20Widgets/dxValidationGroup/3%20Methods/reset().md '/Documentation/ApiReference/UI_Widgets/dxValidationGroup/Methods/#reset') method for this group.
