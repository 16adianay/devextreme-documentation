##### shortDescription
Specifies a color for both the target and zero level lines.

---
This option supports the following colors.

* Hexadecimal colors
* RGB colors
* RGBA colors
* Predefined/cross-browser color names
* [Predefined SVG colors](https://www.w3.org/TR/SVG/types.html#ColorKeywords)

To specify a color for the bullet bar, use the [color](/api-reference/20%20Data%20Visualization%20Widgets/55%20dxBullet/1%20Configuration/color.md '/Documentation/ApiReference/Data_Visualization_Widgets/dxBullet/Configuration/#color') option.
