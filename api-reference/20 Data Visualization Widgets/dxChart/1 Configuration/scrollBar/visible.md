##### shortDescription
Specifies whether the scroll bar is visible or not.

---
[note]Making the scroll bar visible can be useful only if [scrolling is enabled](/api-reference/20%20Data%20Visualization%20Widgets/dxChart/1%20Configuration/scrollingMode.md '/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/#scrollingMode') in your chart.
