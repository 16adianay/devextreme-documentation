##### shortDescription
Specifies the color of the series.

---
This option supports the following colors.

* Hexadecimal colors
* RGB colors
* RGBA colors
* Predefined/cross-browser color names
* [Predefined SVG colors](https://www.w3.org/TR/SVG/types.html#ColorKeywords)

Specified in the [commonSeriesSettings](/api-reference/20%20Data%20Visualization%20Widgets/dxChart/1%20Configuration/commonSeriesSettings '/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/commonSeriesSettings/') object, this option colors all series in the chart. To color an individual series, specify this option in the series object within the [series](/api-reference/20%20Data%20Visualization%20Widgets/dxChart/1%20Configuration/series '/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/series/') array.
