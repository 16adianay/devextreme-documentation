##### shortDescription
Exports the widget into a document with a specified name and format.

##### param(fileName): string
The name of a destination file.

##### param(format): string
A required format.<br/>
Accepted Values: 'PNG' | 'PDF' | 'JPEG' | 'SVG' | 'GIF'
