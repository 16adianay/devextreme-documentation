---
##### shortDescription
Occurs in **Chart**when two or more value axes on a single pane have the same name.

---
When you visualize data using several value axes on a single [pane](/concepts/20%20Data%20Visualization/10%20Charts/10%20Chart%20Elements/110%20Panes.md '/Documentation/Guide/Data_Visualization/Charts/Chart_Elements/#Panes'), these axes must have different [names](/api-reference/20%20Data%20Visualization%20Widgets/10%20dxChart/1%20Configuration/valueAxis/name.md '/Documentation/ApiReference/Data_Visualization_Widgets/dxChart/Configuration/valueAxis/#name'). If they do not, the E2102 occurs.
