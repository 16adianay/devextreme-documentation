##### shortDescription
Disables the [zooming](/concepts/20%20Data%20Visualization/35%20VectorMap/40%20End-User%20Interaction/20%20Zooming/10%20Zooming.md '/Documentation/Guide/Data_Visualization/VectorMap/End-User_Interaction/#Zooming') capability.

---
Setting this option to *false* disables a user's ability to zoom the map. However, you can still zoom the map in code using the [zoomFactor(zoomFactor)](/api-reference/20%20Data%20Visualization%20Widgets/70%20dxVectorMap/3%20Methods/zoomFactor(zoomFactor).md '/Documentation/ApiReference/Data_Visualization_Widgets/dxVectorMap/Methods/#zoomFactorzoomFactor') method.
