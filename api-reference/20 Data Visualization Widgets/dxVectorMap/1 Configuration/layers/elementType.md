##### notUsedInTheme

##### shortDescription
Specifies the type of a marker element. Setting this option makes sense only if the [layer type](/api-reference/20%20Data%20Visualization%20Widgets/dxVectorMap/1%20Configuration/layers/type.md '/Documentation/ApiReference/Data_Visualization_Widgets/dxVectorMap/Configuration/layers/#type') is *"marker"*.

---
When using the widget as an [ASP.NET MVC Control](/concepts/35%20ASP.NET%20MVC%20Controls/20%20Fundamentals '/Documentation/Guide/ASP.NET_MVC_Controls/Fundamentals/'), specify this option using the `VectorMapMarkerType` enum. This enum accepts the following values: `Dot`, `Bubble`, `Pie` and `Image`.
