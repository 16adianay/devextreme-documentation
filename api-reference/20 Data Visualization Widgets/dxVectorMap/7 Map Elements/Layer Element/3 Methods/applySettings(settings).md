---
##### shortDescription
Applies the layer element settings and updates element appearance.

##### param(settings): Object
A <a href="/Documentation/16_2/ApiReference/Data_Visualization_Widgets/dxVectorMap/Configuration/layers/">configuration object</a> for the element.

---
The **settings** object should contain one or more options to be changed for the layer element. The options available to change using this method are listed in the [layers](/api-reference/20%20Data%20Visualization%20Widgets/dxVectorMap/1%20Configuration/layers '/Documentation/ApiReference/Data_Visualization_Widgets/dxVectorMap/Configuration/layers/') option description.

Note that you cannot change the [dataSource](/api-reference/20%20Data%20Visualization%20Widgets/dxVectorMap/1%20Configuration/layers/dataSource.md '/Documentation/ApiReference/Data_Visualization_Widgets/dxVectorMap/Configuration/layers/#dataSource') option for a separate layer element.
