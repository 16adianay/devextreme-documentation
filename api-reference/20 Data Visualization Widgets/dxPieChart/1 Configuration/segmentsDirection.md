##### shortDescription
Specifies the direction that the pie chart segments will occupy.

---
When configuring the widget using [ASP.NET MVC Wrappers](/concepts/35%20ASP.NET%20MVC%20Wrappers/20%20Fundamentals '/Documentation/Guide/ASP.NET_MVC_Wrappers/Fundamentals/'), specify this option using the `PieChartSegmentsDirection` enum. This enum accepts the following values: `Clockwise` and `AntiClockwise`.
