##### shortDescription
Specifies a border color for point labels.

---
This option supports the following colors.

* Hexadecimal colors
* RGB colors
* RGBA colors
* Predefined/cross-browser color names
* [Predefined SVG colors](https://www.w3.org/TR/SVG/types.html#ColorKeywords)
